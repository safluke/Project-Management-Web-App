Background

I undertook this capstone project over 5 days to bring together the following concepts:

Back-End Development

  • Java
  • Spring Boot
  • Controller-Service-Repo-Entity patterns with DTO access
  • Maven
  • Entity-Relationships
  • MySQL

Front-End Development

  • CSS
  • HTML
  • javascript
  • REST APIs
  • Authentication and Authorization with Back-End

I made extensive use of agile methodologies for this project. Gitbash was employed to push code to repository branches, as well as the use of Management mechanisms such as kanban boards.
